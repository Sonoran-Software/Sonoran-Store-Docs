# 🧣 Bandanas

Add more movement and realism to your FiveM clothing lineup with this pocket bandana EUP accessory pack.

This pack includes scratch-made bandanas designed to hang naturally from your character’s pocket. Unlike static accessories, these bandanas include physics, allowing them to move and flow as your character walks, runs, and turns. It’s a simple detail that adds more life to roleplay outfits, gang clothing, civilian fits, and custom EUP setups.

Built with performance and flexibility in mind, the pack includes 8 unisex positions and 26 texture options, making it easy to use across a wide range of outfits and server styles.

Features:

* 8 unisex positions
* 26 texture variations
* Physics-enabled movement
* Naturally flows while walking and moving
* Optimized for FiveM
* Scratch-made original model
* No premade models or ripped assets
* 2,169 polygons
* Escrowed for secure distribution

Perfect for servers looking to add more realistic, customizable clothing accessories without sacrificing performance.
